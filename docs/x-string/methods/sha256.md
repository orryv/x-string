## Technical details

**Signature:**

```php
public function sha256(bool $raw_output = false): self
```

| Namespace | Instance / Static | Immutable (returns clone) | Public / Private / Protected |
| --- | --- | --- | --- |
| `Orryv\XString` | Instance | ✓ | Public |

## Description

Calculates the SHA-256 hash of the string. By default a 64-character hexadecimal digest is produced. When `$raw_output` is `true`
the raw 32-byte binary hash is returned instead.

## Important notes and considerations

- **Immutable operation.** The original instance is not modified.
- **Strong cryptographic digest.** SHA-256 is suitable for integrity checks and signatures, but remember to include a salt when storing passwords.
- **Binary output may contain null bytes.** Treat it as opaque data.

## Parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `$raw_output` | `bool` | `false` | When `true`, return the 32-byte binary hash instead of hex characters. |

## Returns

| Return Type | Immutable (returns clone) | Description |
| --- | --- | --- |
| `self` | ✓ | `XString` containing the SHA-256 digest (hex or raw bytes). |

## Examples

### Default hexadecimal digest

<!-- test:sha256-hex -->
```php
use Orryv\XString;

$result = XString::new('password')->sha256();

#Test: self::assertSame('5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8', (string) $result);
```

### Raw binary output

<!-- test:sha256-raw -->
```php
use Orryv\XString;

$result = XString::new('password')->sha256(true);

#Test: self::assertSame(32, strlen((string) $result));
#Test: self::assertSame(hash('sha256', 'password', true), (string) $result);
```

### Different inputs yield unique hashes

<!-- test:sha256-different -->
```php
use Orryv\XString;

$first = XString::new('alpha');
$second = XString::new('beta');

#Test: self::assertNotSame((string) $first->sha256(), (string) $second->sha256());
```

### Original instance remains unchanged

<!-- test:sha256-immutable -->
```php
use Orryv\XString;

$value = XString::new('immutable');
$value->sha256();

#Test: self::assertSame('immutable', (string) $value);
```
