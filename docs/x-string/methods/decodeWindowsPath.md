## Technical details

**Signature:**

```php
public function decodeWindowsPath(): self
```

| Namespace | Instance / Static | Immutable (returns clone) | Public / Private / Protected |
| --- | --- | --- | --- |
| `Orryv\XString` | Instance | ✓ | Public |

## Description

Decode Windows-safe paths generated by [`encodeWindowsPath()`](encodeWindowsPath.md). The method preserves drive letters and UNC
roots while decoding each percent-encoded segment back to its original characters, including reserved device names and trailing
whitespace.

## Important notes and considerations

- **Per-segment decoding.** Each component between `\\` separators is decoded individually.
- **Round-trip symmetry.** Paths produced by `encodeWindowsPath()` return to their original spelling when decoded.
- **No validation.** The method decodes `%XX` escapes but does not sanitise the resulting path.

## Returns

| Return Type | Immutable (returns clone) | Description |
| --- | --- | --- |
| `self` | ✓ | Original Windows path restored from a percent-encoded representation. |

## Examples

### Decode forbidden characters per segment

<!-- test:windows-decode-path-forbidden -->
```php
use Orryv\XString;

$value = XString::new('C:\\logs\\error%3F.txt');
$result = $value->decodeWindowsPath();

#Test: self::assertSame('C:\\logs\\error?.txt', (string) $result);
```

### Restore reserved device names

<!-- test:windows-decode-path-reserved -->
```php
use Orryv\XString;

$value = XString::new('\\\\server\\share\\%61ux');
$result = $value->decodeWindowsPath();

#Test: self::assertSame('\\\\server\\share\\aux', (string) $result);
```

### Recover trailing whitespace

<!-- test:windows-decode-path-trailing -->
```php
use Orryv\XString;

$value = XString::new('C:\\data%20\\');
$result = $value->decodeWindowsPath();

#Test: self::assertSame('C:\\data \\', (string) $result);
```

### Decode escaped percent signs

<!-- test:windows-decode-path-percent -->
```php
use Orryv\XString;

$value = XString::new('D:\\reports\\100%25 ready');
$result = $value->decodeWindowsPath();

#Test: self::assertSame('D:\\reports\\100% ready', (string) $result);
```
