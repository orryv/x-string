## Technical details

**Signature:**

```php
public function decodeSafePath(): self
```

| Namespace | Instance / Static | Immutable (returns clone) | Public / Private / Protected |
| --- | --- | --- | --- |
| `Orryv\XString` | Instance | ✓ | Public |

## Description

Decode cross-platform safe paths generated by [`encodeSafePath()`](encodeSafePath.md). Percent-encoded segments are restored to
their original characters while `/` separators remain unchanged.

## Important notes and considerations

- **Separator normalisation preserved.** Decoding keeps the `/` separators introduced during encoding.
- **Round-trip safe.** Paths encoded by `encodeSafePath()` decode back to the original spelling (with `/` separators).

## Returns

| Return Type | Immutable (returns clone) | Description |
| --- | --- | --- |
| `self` | ✓ | Decoded portable path string. |

## Examples

### Restore drive prefixes

<!-- test:safe-decode-path-drive -->
```php
use Orryv\XString;

$value = XString::new('C%3A/logs/error%3F.txt');
$result = $value->decodeSafePath();

#Test: self::assertSame('C:/logs/error?.txt', (string) $result);
```

### Restore UNC-style roots

<!-- test:safe-decode-path-unc -->
```php
use Orryv\XString;

$value = XString::new('//server/share/%41UX');
$result = $value->decodeSafePath();

#Test: self::assertSame('//server/share/AUX', (string) $result);
```

### Preserve trailing separators

<!-- test:safe-decode-path-trailing -->
```php
use Orryv\XString;

$value = XString::new('workspace/');
$result = $value->decodeSafePath();

#Test: self::assertSame('workspace/', (string) $result);
```

### Decode percent signs

<!-- test:safe-decode-path-percent -->
```php
use Orryv\XString;

$value = XString::new('reports/100%25 ready');
$result = $value->decodeSafePath();

#Test: self::assertSame('reports/100% ready', (string) $result);
```
