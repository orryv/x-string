## Technical details

**Signature:**

```php
public function passwordVerify(string $hash): bool
```

| Namespace | Instance / Static | Immutable (returns clone) | Public / Private / Protected |
| --- | --- | --- | --- |
| `Orryv\\XString` | Instance | — (returns `bool`) | Public |

## Description

Verifies the current string against a password hash using PHP's `password_verify()` function.

## Important notes and considerations

- **Constant-time comparison.** Relies on `password_verify()`, which mitigates timing attacks.
- **No mutation.** The `XString` instance is not modified.
- **Algorithm agnostic.** Works with any hash generated by `password_hash()` or compatible tooling.

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `$hash` | `string` | The stored hash to compare against. |

## Returns

| Return Type | Description |
| --- | --- |
| `bool` | `true` when the string matches the hash, `false` otherwise. |

## Thrown exceptions

`—` This method does not throw additional exceptions beyond those raised internally by PHP (none for `password_verify()`).

## Examples

### Verify a matching bcrypt hash

<!-- test:password-verify-match -->
```php
use Orryv\XString;

$password = XString::new('secret');
$hash = $password->passwordHash();

$result = XString::new('secret')->passwordVerify((string) $hash);

#Test: self::assertTrue($result);
```

### Mismatched password yields false

<!-- test:password-verify-mismatch -->
```php
use Orryv\XString;

$hash = XString::new('correct-horse')->passwordHash();

$result = XString::new('Tr0ub4dor&3')->passwordVerify((string) $hash);

#Test: self::assertFalse($result);
```

### Invalid hash formats fail gracefully

<!-- test:password-verify-invalid -->
```php
use Orryv\XString;

$result = XString::new('secret')->passwordVerify('not-a-real-hash');

#Test: self::assertFalse($result);
```

### Original instance is untouched

<!-- test:password-verify-immutability -->
```php
use Orryv\XString;

$value = XString::new('immutable');
$value->passwordVerify('irrelevant');

#Test: self::assertSame('immutable', (string) $value);
```
